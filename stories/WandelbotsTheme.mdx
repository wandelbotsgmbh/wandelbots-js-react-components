# Wandelbots MUI Theme

The component library comes with a Wandelbots theme for Material UI which is the default theme used in this storybook. 
If you like, you can use it in your own app to give the components a similar look.
The theme is accessed by calling `createNovaTheme`, and can be [customized](?path=/docs/theming-theming-components--docs) as needed.

The theme currently supports **dark mode only**. Passing `palette: { mode: "light" }` is not supported yet (it throws upstream).

> **Migration note:** `createNovaMuiTheme` is deprecated. Rename to `createNovaTheme` — it accepts
> the same options (now variadic: pass multiple `ThemeOptions` objects to deep-merge in order).
> The old name will be removed in the next major release.

<Source dark={true} code={`
import { ThemeProvider } from "@mui/material"
import { createNovaTheme } from "@wandelbots/wandelbots-js-react-components/core"

const theme = createNovaTheme()

<ThemeProvider theme={theme}>
  ...
</ThemeProvider>
`}/>
