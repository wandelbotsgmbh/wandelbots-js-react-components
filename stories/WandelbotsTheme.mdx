# Wandelbots MUI Theme

This page shows how various regular MUI components look with the default Wandelbots MUI theme.
You can use also [customize or replace](?path=/docs/theming-theming-components--docs) this theme.

<ThemeProvider theme={theme}>
  ...
</ThemeProvider>
