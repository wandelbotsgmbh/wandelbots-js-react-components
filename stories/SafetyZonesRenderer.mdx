# Safety Zones Renderer (API v2)

The `SafetyZonesRenderer` component visualizes the safety zones of the controller.

This component should be used with [nova-js](https://github.com/wandelbotsgmbh/nova-js). Pass the `safety_zones` from the `motion group description` (API v2) to the component.

```tsx
<SafetyZonesRenderer safetyZones={motionGroupDescription.safety_zones || {}} />
```

### Optional: `dhParameters`

For plane safety zones, the rendered plane size depends on the robot's reach radius. Pass `dhParameters` from the motion group description to correctly scale plane safety zones based on the robot's kinematics. Without this prop, planes fall back to a fixed default size.

```tsx
<SafetyZonesRenderer
  safetyZones={motionGroupDescription.safety_zones || {}}
  dhParameters={motionGroupDescription.dh_parameters}
/>
```

# Safety Zones Renderer (API v1)

The `SafetyZonesRenderer` component visualizes the safety zones of the controller.

This component should be used with [nova-js](https://github.com/wandelbotsgmbh/nova-js). Pass the `safetyZones` from the `connectedMotionGroup` (API v1) to the component.

```tsx
<SafetyZonesRenderer safetyZones={connectedMotionGroup.safetyZones || []} />
```
