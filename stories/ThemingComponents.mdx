# Theming components

The UI components used in this library will respect the [Material UI theme](https://mui.com/material-ui/customization/theming/) of 
the application they are rendered within. So you can customize the look and feel of the components using the MUI theming system.

## Using the Wandelbots default theme

To make the components look like they do in this documentation, apply the provided Wandelbots MUI theme.

<Source dark={true} code={`
import { createNovaMuiTheme, JoggingPanel } from "@wandelbots/wandelbots-js-react-components"

const theme = createNovaMuiTheme()

<ThemeProvider theme={theme}>
  <JoggingPanel ... />
</ThemeProvider>
`}/>

<Canvas of={JoggingStory} sourceState="none"/>

## Extending the default theme

You can also use the Wandelbots theme as a base and extend it with your own customizations, or even merge it
with another theme using MUI [theme composition](https://mui.com/material-ui/customization/theming/#api).

<Source dark={true} code={`
import { createNovaMuiTheme, JoggingPanel } from "@wandelbots/wandelbots-js-react-components"

const theme = createNovaMuiTheme({
  palette: {
    primary: {
      main: "#ff0000"
    }
  }
})

<ThemeProvider theme={theme}>
  <JoggingPanel ... />
</ThemeProvider>
`}/>

<Canvas of={JoggingStoryExtendedTheme} sourceState="none"/>

## Use your own theme

You could instead use an entirely different MUI theme and style the components as you like.

<Source dark={true} code={`
import { createTheme } from '@mui/material/styles';
import { JoggingPanel } from "@wandelbots/wandelbots-js-react-components"

const theme = createTheme({
  typography: {
    fontFamily: 'Comic Sans MS'
  }
})

<ThemeProvider theme={theme}>
  <JoggingPanel ... />
</ThemeProvider>
`}/>

<Canvas of={JoggingStoryReplacedTheme} sourceState="none"/>
