# Jogging Panel

The `JoggingPanel` is a high-level user interface for manually moving a robot using the Wandelbots stack. It needs only a `Nova` instance from [@wandelbots/nova-js/v2](https://github.com/wandelbotsgmbh/nova-js) and the id of a motion group to connect to.
It can be used to quickly create a jogging panel to control a robot. It `automatically detects` the `type of Joint` and displays the correct controls for it. Joints can rotate or translate. The `JoggingStore` uses the `inverseSolver` and `jointType` fetched from the API to determine the correct jogging controls and tabs.

```tsx
import { Nova } from "@wandelbots/nova-js/v2"
import { ConnectedMotionGroup } from "@wandelbots/wandelbots-js-react-components/core"

const connectedMotionGroup = ConnectedMotionGroup.connect(nova, motionGroupId)

<JoggingPanel
  nova={wandelApp.nova}
  motionGroupId={connectedMotionGroup.motionGroupId}
/>
```
